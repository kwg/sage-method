# Browse Assets

Search the asset catalog to find UI elements.

## Steps

1. Load `assets/catalog/asset-catalog.json`
2. Ask user what they're looking for (category, style, purpose)
3. Search by category and tags
4. Present matching packs with:
   - Pack name and description
   - Asset count and formats
   - Sample assets from pack
   - Full NAS path
5. Offer to show more details or search with different criteria

## Reference

**Categories:** button, panel, icon, card, background, portrait, progress, decoration

**Tags:** fantasy, wooden, sci-fi, pixel, cute, dark, minimalist, medieval, modern
