# Excalidraw JSON Validation Instructions

**Status**: Not Available in SAGE Standalone Mode

This file would contain instructions for validating Excalidraw JSON format.

## Purpose

Validates that Excalidraw `.excalidraw` files:
- Have correct JSON structure
- Include required fields
- Use valid element types
- Follow Excalidraw schema

## SAGE Alternative

Use standard JSON validation tools or Excalidraw's built-in validation.
